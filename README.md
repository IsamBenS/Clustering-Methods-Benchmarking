# Clustering Methods Benchmarking
Shiny app allowing the user to evaluate the F score for all the populations within a Reference File used as Input by comparing it to an output file containing clusters generated by a clustering or overclustering algorithm.
 
	
## Requirements
  * software: R(Version 3.4.3 to 3.5), Rstudio(optional)
  * R packages: shiny, shinyjs, ggcyto, flowCore, gtools, Biobase , Rcpp, RcppArmadillo, microbenchmark
  
## Quick installation guide

  1. Run the following command in R/RStudio:
```
install.packages(c("shiny", "shinyjs", "gtools", "ggplot2", "Rcpp", "RcppArmadillo", "devtools", "microbenchmark"))
source("https://bioconductor.org/biocLite.R")
biocLite("Biobase")
biocLite("flowCore")
biocLite("ggcyto")
```
  >You may be asked to reload your environment, if so, accept.
  
  2. Run the next commands:
```
library("devtools")
install_github("isambens/fcsgenerator")
```

  
## Launching the shiny application

  1. Run the following commands in R/RStudio:
```
library("FCSGenerator")
FCSGenerator.run()
```
